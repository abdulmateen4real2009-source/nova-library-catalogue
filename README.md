# 📚 NOVA Library Catalogue

A modern library catalogue system built with Flask and SQLite.

## Features

- Add books
- Search books
- Delete books
- Upload PDF/eBooks
- View PDFs in browser
- Statistics dashboard
- Glassmorphism UI

## Technologies

- Python
- Flask
- SQLite
- HTML
- CSS

## Installation

```bash
git clone <repository-url>
cd library_catalogue

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python database.py
python app.py
```

Open:

http://127.0.0.1:5000
